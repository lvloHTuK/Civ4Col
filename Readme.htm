УСТАНОВКА И ЗАПУСК

Установка Civilization IV: Colonization!

Перед установкой игы          

Перед тем, как установить игру Sid Meier’s Civilization IV: Colonization, пожалуйста, убедитесь, что ваш компьютер соответствует минимальным требованиям, указанным выше.

 

Мы также рекомендуем вам для подготовки жесткого диска к установке предварительно провести сканирование и дефрагментацию диска. Эти утилиты входят в состав Windows® и способны обнаружить и устранить проблемы на вашем жестком диске, которые могли бы помешать инсталляции и/или игровому процессу.

Для запуска этих программ и оптимизации жесткого диска:

  1. Открыть "Мой компьютер" ("My Computer").
  2. Щелкнуть правой кнопкой мыши по диску, на который вы хотите установить игру (обычно C:).
  3. Щелкнуть по "Свойства" ("Properties").
  4. Выбрать в меню "Инструменты" ("Tools").
  5. Щелкнуть "Проверить" ("Check Now") для запуска программы ScanDisk.
  6. После завершения сканирования жесткого диска, повторите шаги 1-3 и выберите "Дефрагментировать" ("Defragment Now") для запуска программы Disk Defragmenter.

Учтите, что дефрагментация диска обычно занимает много времени, так что имеет смысл запустить программу Disk Defragmenter вечером и оставить на ночь.

Установка игры

Чтобы установить Civilization IV: Colonization,

  1. Вставьте диск с игрой Sid Meier’s Civilization IV: Colonization в DVD-ROM привод..
  2. Если на вашей системе включен автозапуск, появится окно автозапуска. Выберите кнопку "УСТАНОВИТЬ". Если, когда вы вставите диск в дисковод, ничего не произойдет, откройте "Мой компьютер" ("My Computer"), дважды щелкните мышью по букве дисковода DVD-ROM, затем дважды щелкните по Setup.exe.
  3. Когда начнется процесс установки, выполняйте указания на экране, чтобы установить игру.

Запуск игры

По завершению процесса установки, выберите в меню автозапуска "Играть". Для того, чтобы начать игру, диск с игрой должен находиться в дисководе.

Права пользователя

Вы должны иметь привелегии Администратора, чтобы установить Civilization IV: Colonization .

Поиск/установка DirectX®

Во время установки Civilization IV: Colonization, ваша система будет просканирована, чтобы выяснить, установлена ли на вашем компьютере последняя версия DirectX®, которая содержиться на диске Sid Meier's Civilization IV: Colonization. Если нужная версия DirectX® не будет обнаружена, произойдет автоматическое обновление вашей версии.

 

 

Запуск игры

 Есть несколько способов начать игру.

  1. Вставьте диск Sid Meier’s Civilization IV: Colonization в привод DVD-ROM. После появления программы автозапуска, нажмите на кнопку "Играть".
  2. Если вы отметили пункт, что хотите создать на рабочем столе ярлык для игры, то вы можете начать игру, щелкнув по ярлыку дважды левой кнопкой мыши (на компьютере с установленной Vista можно получить доступ к игре через игровое меню Explorer).
  3. Нажмите Пуск (Start) -> Программы (Programs) -> 1C -> 2K Games -> Firaxis Games -> Sid Meier’s Civilization 4: Colonization и выберите Sid Meier’s Civilization IV: Colonization

 

Решение проблем связанных с системой

Оптимизация игрового процесса на медленных компьютерах

Если на вашем компьютере игра идет слишком медленно, попробуйте снизить до минимума уровень прорисовки деталей, а также другие установки графики, а затем настраивайте их по одной, пока не достигнете оптимального баланса между качеством изображения и игровыми показателями. Установки графики можно настроить, нажав [Ctrl – 0], либо через меню "Настройка" в главном меню.

Проблемы с графикой

Основные проблемы с графикой

Устаревшие или неподходящие видеодрайверы могут стать причиной большинства проблем с графикой. Прежде чем связаться со службой поддержки, проверьте, установлен ли у вас самый последний из драйверов для вашей видеокарты.

  • ATI Technologies
  • Intel
  • Matrox
  • NVIDIA
  • S3 Graphics
  • SiS

Мною получено сообщение, где говорится: "Unable to initialize DirectX", при попытке запустить игру.

Такое чаще всего происходит на компьютерах с графической картой ATI. Пожалуйста, пройдите по этой ссылке, чтобы решить данную проблему: http://www.2kgames.com/civ4/support.htm

 

Проблемы со звуком

Если у вас появились проблемы со звуком, включая проблемы с громкостью, отсутствием аудиоканалов (например, звук не идет через центральный канал колонок), обрывы, либо любые иные затруднения, мы рекомендуем подключить аудиодрайвер, установленный по умолчанию: MILES Fast 2D Driver.

 

 

Проблемы с сетью

Настоятельно рекомендуется высокоскоростное подключение к Интернету.

 

Если у вас есть проблемы с асинхронностью (OOS) в ходе сетевых игр, когда игроки по ходу подключаются к игре, управляемой ИИ, либо покидают игру, рекомендуется, чтобы хозяин игры сохранил ее и перезапустил из главного меню.

 

Если игрок непреднамеренно выбыл из сетевой игры, всем остальным рекомендуется дождаться его возвращения на экране Демократии, чтобы он не подключался к игре по ходу.

 

Хотя в сетевых играх может принимать участие до 18 игроков, мы поддерживаем и рекомендуем максимально 12 участников.

 

 

Сохранение игры

Сохраненные игры записываются в особую папку, в директории "Мои документы" ("My Documents") текущего пользователя. Найти эту папку можно, если перейти в "Мои документы" ("My Documents"), затем в "Мои игры" ("My Games"), и наконец оттуда в "Colonization".

Сохраненные игрыCivilization IV: Colonization имеют расширение ".ColonizationSave", а сохраненные файлы Worldbuilder имеют расширение “.ColonizationWBSave”. Вы можете копировать или стирать эти файлы так же, как любые другие. Не рекомендуется переименовывать сохраненные игры, поскольку программа способна загружать только определенные виды файлов.

Дважды щелкните мышью по сохраненному файлу, чтобы загрузить и запустить игру.

 

Файл INI

Продвинутые пользователи могут изменять различные настройки и параметры Civilization IV: Colonization редактируя файл INI. Этот файл можно найти в директории "Мои документы" ("My Documents"), "Мои игры" ("My Games"), "Colonizationv, после чего следует открыть файл “CivilizationIV.ini”. Этот файл является обычным текстовым файлом, который можно редактировать с помощью программы Блокнот (Notepad), входящей в состав Windows®.

Файл Civilization4.ini содержит в себе параметры всех установок, которые может изменять пользователь.
